Medidas de dispersão

Definição

As medidas de posição na estatística são a média, moda e mediana. Elas indicam a tendência central dos dados e suas devidas posições em relação ao rol. São utilizadas para compreensão e análise de uma determinado conjunto de dados.
Média

A média é calculada através da soma de todos os elementos do conjunto dividida pela quantidade de elementos (n). Sua fórmula para um conjunto de dados em rol é a acima.
Moda
A moda é simplesmente o elemento com a maior frequência dentro do conjunto. Para descobri-la, é apenas necessário observar e comparar a quantidade de elementos. Quando um conjunto possui 1 moda, é chamado de unimodal, quando possui mais de 1 moda, é chamado de multimodal. Se forem especialmente 2, é chamado de bimodal.
Mediana
A mediana é o centro físico ou o valor central da distribuição dos elementos (quando em rol). Quando o conjunto possuir um número ímpar de elementos, será simplesmente o elemento central (ou, em outras palavras, o elemento de posição n/2, arredondado para cima, onde n é a quantidade de elementos). Quando o conjunto possuir um número par de elementos, a mediana será a média dos 2 valores centrais.
